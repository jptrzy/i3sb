# i3sb
I3 Simple Bar is as in the name, simple bar to i3 window manager, that alsow suport pywal.

## Installation

**Manual/Git install**
```
git clone https://github.com/jptrzy/i3sb
cd i3sb
make
sudo make install
```

## Customization
You could customize your bar via editing `config.h` and then compiling i3sb.
